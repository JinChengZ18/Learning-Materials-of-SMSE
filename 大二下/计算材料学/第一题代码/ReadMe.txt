安装步骤：
(1)安装python3.X，注意勾选添加python到系统环境变量；
(2)安装pip（一般python已经自带了，在windows命令行输入python；import pip；若无报错则有）；
(3)windows命令行输入 pip install -i https://pypi.tuna.tsinghua.edu.cn/simple numpy 可安装numpy包，其他包类似

python编译器安装：
推荐百度搜索 geany，比较好用;
打开py文件后按F5可编译并运行;
如果不能执行,则点击geany任务栏上端生成--->设置生成命令; 配置可百度
